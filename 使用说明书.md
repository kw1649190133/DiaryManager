## 1. 产品简介

「我的日记本」是一款轻量级本地日记管理工具，所有数据以纯文本格式保存在本地，无需联网、无需账号，隐私完全由你掌控。

**核心特色：**

- 纯文本存储，数据透明可控，可用任何文本编辑器直接打开
- 按年份自动归档，文件结构清晰
- 可视化日历，一眼看出哪些天写了日记
- 自动保存，写完即走，不用担心丢失
- 支持浅色/深色双主题
- 支持字体大小调节

## 2. 安装与启动

### 方式一：独立可执行文件（推荐）

1. 将 `DiaryManager` 文件夹复制到任意位置
2. 双击 `DiaryManager.exe` 即可启动

> 整个文件夹需保持完整，不要单独移动 exe 文件。

### 方式二：从源码运行

需要 Python 3.10+ 和 PySide6。

```bash
# 安装依赖
pip install PySide6

# 启动图形界面
python main.py -g
```

## 4. 日历面板

日历面板是日记导航的核心，位于窗口左侧。

### 4.1 日期标记含义

| 标记样式 | 含义 |
|---------|------|
| **绿色圆点** | 该日期有日记 |
| **蓝色实心圆** | 当前选中的日期 |
| **蓝色边框圆** | 今天的日期（无日记时） |
| **绿底蓝边圆** | 今天且有日记 |
| **无标记** | 该日期无日记 |

### 4.2 月份导航

- 点击 **◀** 按钮：切换到上一月
- 点击 **▶** 按钮：切换到下一月

### 4.3 年份快速跳转

日历顶部提供年份导航栏：

| 按钮 | 功能 |
|------|------|
| `<<` | 后退 10 年 |
| `<` | 后退 1 年 |
| `>` | 前进 1 年 |
| `>>` | 前进 10 年 |
| 年份输入框 | 直接输入目标年份，按回车跳转（支持 1900–2100） |

### 4.4 月度信息栏

日历下方显示当月统计，例如：

> 本月写了 15 天 | 未写 16 天

### 4.5 点击日期

- 点击日历上的任意日期 → 右侧编辑器跳转到该日期
- **不会自动创建日记文件**，仅加载显示
- 若该日期无日记，编辑器显示空白内容，状态显示"新建日记"

## 5. 编辑日记

### 5.1 编辑区域

右侧编辑器是纯文本编辑区域，不支持富文本格式（无加粗、无图片）。这种设计保证所有日记都是纯文本文件，可用任何编辑器打开。

### 5.2 字体大小调节

编辑器右上角提供字体控制：

| 操作 | 方式 |
|------|------|
| 放大字体 | 点击 **A+** 按钮，或菜单"视图 → 放大字体"，或快捷键 `Ctrl+=` |
| 缩小字体 | 点击 **A-** 按钮，或菜单"视图 → 缩小字体"，或快捷键 `Ctrl+-` |

字体大小范围：10pt – 32pt，默认 14pt。

### 5.3 字数统计

- 编辑器底部状态栏实时显示当前字数
- 主窗口底部状态栏也显示字数
- 字数统计以纯文本去除首尾空白后的字符数计算

### 5.4 编辑器状态指示

编辑器底部右侧显示当前日记状态：

| 状态文字 | 含义 |
|---------|------|
| 已有日记 | 该日期存在日记文件 |
| 新建日记 | 该日期尚无日记文件 |

主窗口底部状态栏用 `*` 标记未保存的更改：

> 编辑中: 2026-05-25 (128字) *

## 6. 保存与自动保存

### 6.1 手动保存

- 点击工具栏 **保存** 按钮
- 菜单栏 **文件 → 保存**
- 快捷键 `Ctrl+S`

### 6.2 自动保存

程序在以下时机自动保存当前编辑内容：

1. **切换日期时**：点击日历上的其他日期前，自动保存当前内容
2. **关闭程序时**：退出程序前，自动保存当前内容

### 6.3 空内容不保存

如果编辑器内容为空白（仅空格、换行），自动保存不会创建文件。这样可以避免为没有日记的日期产生空文件。

> 如果你写了一些内容后全部删除，程序会认为内容为空，不会保存空文件。

## 7. 新建与删除日记

### 7.1 新建日记

1. 先在日历上点击目标日期
2. 点击工具栏 **新建日记** 按钮
3. 程序为该日期创建日记文件

**注意：**

- 新建日记针对的是当前日历选中的日期
- 如果该日期已有日记，会弹出提示"日记已存在"，不会覆盖
- 如果没有选中任何日期，则默认为今天新建

### 7.2 删除日记

1. 在日历上点击要删除的日期
2. 点击工具栏 **删除** 按钮
3. 弹出确认对话框，点击 **是** 确认删除

**注意：**

- 删除操作不可撤销，请确认后再操作
- 删除后编辑器会清空内容，但保留日期标题
- 删除后日历上的绿色标记会立即消失
- 如果当前日期没有日记，会提示"当前日期没有日记可删除"

## 8. 搜索日记

### 8.1 使用搜索

1. 点击工具栏 **搜索** 按钮
2. 在弹出的对话框中输入关键词
3. 点击确定，显示搜索结果列表

### 8.2 搜索结果

- 结果以列表形式展示，显示日期和字数
- **双击**某条结果即可跳转到该日期查看
- 搜索不区分大小写
- 搜索范围覆盖所有日记的全部内容

## 12. 数据存储说明

### 12.1 文件结构

所有日记数据存储在 `data/` 目录下，按年份自动归档：

```
data/
├── 2024/
│   ├── 20240101.txt
│   ├── 20240115.txt
│   └── 20241231.txt
├── 2025/
│   ├── 20250301.txt
│   └── 20250520.txt
└── 2026/
    └── 20260525.txt
```

### 12.2 文件命名规则

- 文件名格式：`YYYYMMDD.txt`
- 示例：`20260525.txt` 代表 2026 年 5 月 25 日
- 编码：UTF-8

### 12.3 数据安全

- 所有数据均为纯文本，可用记事本等工具直接打开和编辑
- 没有任何加密、云端同步或远程传输
- 建议定期备份 `data/` 文件夹（直接复制即可）
- 删除日记会直接删除对应的 .txt 文件，且不可恢复

### 12.4 自动导入

程序启动时会自动扫描 `data/` 目录，如果发现不在正确年份子目录下的日记文件（如 `data/20240101.txt`），会自动将其移动到 `data/2024/20240101.txt`。

### 12.5 手动导入

你也可以将符合 `YYYYMMDD.txt` 命名规则的文件放入 `data/` 目录，启动程序后会自动识别和归档。

## 13. 常见问题

### Q: 我不小心删除了日记，能恢复吗？

删除操作是直接删除文件的，无法通过程序恢复。如果你有 `data/` 文件夹的备份，可以从备份中恢复对应的 .txt 文件。

### Q: 可以在手机上使用吗？

本程序为 Windows 桌面应用，暂不支持手机端。但日记文件为纯文本格式，你可以将 `data/` 文件夹复制到手机上用文本编辑器查看。

### Q: 日记文件可以用其他编辑器修改吗？

可以。日记文件就是普通的 .txt 文件，用任何文本编辑器打开编辑都不会影响程序读取。修改后回到程序点击 **刷新**（F5）即可同步最新状态。

### Q: 为什么切换日期时不会自动创建日记？

v1.4 版本移除了点击日期自动创建日记的功能。现在只有点击工具栏的 **新建日记** 按钮才会创建文件，避免产生空文件。如果你需要为某个日期写日记，请先选中日期，再点击新建。

### Q: 编辑了内容但没有手动保存，会丢失吗？

不会。程序在切换日期和关闭窗口时会自动保存有内容的编辑。但如果你写了内容后又全部删除清空，程序不会保存空内容。

### Q: 为什么 exe 文件夹有 100 多 MB？

程序打包时包含了 Python 运行时和 PySide6（Qt 框架）的全部依赖库，因此体积较大。这是正常现象，不影响使用性能。建议保持文件夹完整，不要删除其中的文件。

### Q: 可以修改字体类型吗？

当前版本仅支持调节字体大小，暂不支持更换字体。字体默认使用微软雅黑。

### Q: 如何备份数据？

直接复制程序目录下的 `data/` 文件夹即可。备份可存放在 U 盘、网盘或任意安全位置。

# 开发更新日志

## v1.4 (2026-05-25)

### 功能调整

- **移除点击日期自动创建日记的逻辑**
  - 点击日历上无日记的日期，直接跳转查看（编辑器显示空白 + 日期标题）
  - 不再弹出"是否创建"对话框
  - 必须点击工具栏"新建日记"才会真正创建文件

- **"新建日记"行为优化**
  - 若当前有选中日期，则为该日期新建日记
  - 若无选中，则为今天新建
  - 若日记已存在，弹出提示，不覆盖

### 新增功能

- **刷新日历**
  - 工具栏新增"刷新"按钮
  - 菜单栏文件菜单新增"刷新日历"（快捷键 F5）
  - 重新扫描文件系统，同步日历面板与实际文件状态

### Bug 修复

- 删除日记后日历标记不消失 → 删除后立即 `calendar_view.refresh()`
- 删除后编辑器日期标题消失 → 删除后保留日期标题，仅清空内容
- 删除后自动保存重建文件 → 删除后立即 `editor.mark_saved()`

## v1.0 (2026-05-18)

### 首次发布

#### 核心功能

- **日记管理**：创建、编辑、删除、查看日记
- **数据存储**：纯文本格式，按年份分目录存储 (data/YYYY/YYYYMMDD.txt)
- **日历视图**：可视化查看每日日记状态
  - 绿色标记：有日记的日期
  - 蓝色边框：今日
  - 选中高亮：当前查看日期
- **搜索功能**：按关键词搜索日记内容
- **统计功能**：
  - 按日/月/年查看日记数量
  - 字数统计与可视化图表

#### 界面特性

- **双主题支持**：浅色/深色主题一键切换
- **字体缩放**：编辑器支持 Ctrl+鼠标滚轮 / A+/A- 缩放字体
- **程序图标**：自定义 diary.ico 桌面与任务栏图标

#### 技术实现

- **架构**：核心层 (core/) 与 UI 完全分离
- **GUI**：PySide6 现代化图形界面
- **命令行**：完整的 CLI 接口支持
- **打包**：PyInstaller 一键打包为独立可执行文件

#### 启动方式

```bash
# GUI 模式
python main.py -g

# CLI 模式
python main.py list 2026
python main.py stats
python main.py search "关键词"
```
